# 🧮 Advanced Calculator

A responsive and interactive calculator built using **HTML, CSS, and JavaScript**. This project performs real-time calculations with a clean and user-friendly interface, demonstrating core front-end development and JavaScript concepts.

---

## 📖 Overview

The Advanced Calculator is designed to perform basic arithmetic operations with a simple and modern user interface. It provides instant results while maintaining a responsive layout for desktop and mobile devices.

---

## ✨ Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🧹 Clear (AC) functionality
* ⌫ Delete last input
* 📱 Fully responsive design
* ⚡ Real-time calculations
* 🎨 Clean and modern UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
